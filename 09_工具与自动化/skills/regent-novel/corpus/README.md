# 轻量写作语料库

这里保存可复用的写作方法卡、场景模式和公开资料研究笔记，不保存小说正史，也不收录受版权保护作品的长段原文。

## 权威边界

- `corpus/` 的优先级低于仓库内正式设定、大纲、连续性文件和已确认的语言风格文件。
- 当前项目未见独立的正式“语言风格与叙事基调”文件，因此这些卡片只提供低置信度的写作抓手；使用时需服从现有正文和用户指令。
- 场景模式不是情节模板，不要求固定反转、固定爽点或固定章尾悬念。

## 使用方式

按任务只加载一到三张相关卡片。例如，道路旅行场景可读 `style_cards/place-and-road.md` 与 `scene_patterns/road-and-camp.md`；技术场景可读 `style_cards/encyclopedic-integration.md` 与 `scene_patterns/garage-and-technical.md`。
