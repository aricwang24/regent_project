# 技术、空间与装备一致性

覆盖车辆、改装、装备、枪械、狩猎动作、伤病机制、露营系统和空间走位。权威顺序以专项机设高于上位摘要。

## 建立场景状态

- 机器：型号/虚构身份、当前配置、载荷、油电水、故障、维护和可用部件。
- 空间：人物位置、出入口、视线、距离、地形、天气与光照。
- 装备：由谁携带、何时取出、合法/安全前提、弹药或消耗品。
- 身体：伤病部位、疼痛、活动限制、恢复时间和医疗处置。

## 校验方法

1. 先从正式机设读取锁定值，不凭常识覆盖。
2. 画出动作链：准备—执行—反馈—故障/代价—收尾。
3. 检查重量、尺寸、时间、能源、道路条件和人员能力是否相容。
4. 技术细节必须改变选择、风险、劳动或人物关系；无叙事功能的参数应删除或缩写。
5. 现实产品、法规、安全标准、狩猎季和精确性能进入 [事实研究](factual-research.md)。
6. 虚构机器可以超出现实，但必须遵守项目已锁定的内部规则与维护成本。

## 输出

按“已确认 / 需研究 / 与正史冲突 / 空间或动作断裂”报告。提供最小修复方案，不自动重写正式机设。
