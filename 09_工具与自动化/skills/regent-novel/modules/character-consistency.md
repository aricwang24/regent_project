# 人物一致性

用于规划、正文和审稿中的人物校验。目标不是让人物永远理性，而是让偏差有原因、有过程、有后果。

## 六项核对

1. **目标**：人物此刻想得到、保护或逃避什么。
2. **知识**：人物已经知道什么，通过何种渠道知道。
3. **能力**：专业技能、体能、语言、经验和资源是否足够。
4. **状态**：伤病、疲劳、酒精、恐惧、愤怒和环境压力如何影响动作。
5. **声音**：词汇、句长、回避方式、幽默和沉默是否符合其身份与关系。
6. **弧线**：本次变化是否由此前事件积累，而非作者临时需要。

## 判断级别

- `通过`：行为可由既有目标与状态充分解释。
- `需铺垫`：行为可以成立，但正文缺少触发或过渡。
- `冲突`：与权威设定、知识状态或能力边界直接相悖。
- `未知`：项目资料不足，不能替角色补设定。

发现问题时，优先补动机、触发、代价或后续反应；不要偷偷修改人物底层设定。若确需变更正式人物设定，转入 [正史管理](canon-management.md) 并等待明确授权。
