## 项目起源

2022年秋，拉法收来一辆仍可行驶但保养状态一般的1996 Chevrolet Caprice 9C1，并通过试驾促使林峻接手。林峻喜欢其外形、气质与老派驾驶感，却不满原动力表现；拉法随后以手中积压的416发动机和强化4L80E引导项目由整备转向性能改装，Whipple留作后续升级诱因。

本节为当前正式项目起源，取代专项总设定中“拆除原动力总成后作为备件车低价转让”的旧描述；车辆技术参数仍以专项总设定为准。
