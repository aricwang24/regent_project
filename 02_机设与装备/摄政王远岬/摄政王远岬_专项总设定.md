# 一、项目总纲

## 1. 身份与定位

**正式项目名：**

> **GMC Regency Foreland / 摄政王·远岬**

摄政王不是普通露营车，也不是追求岩石攀爬的硬派越野房车，而是一台建立在传统美式全尺寸厢式车基础上的：

> **重型豪华四驱 Grand Touring / Expedition Coach**

它同时承担长途巡航、国家公园旅行、猎场后勤、北境远征、摄影平台、私人影院、移动工作站、宠物安全舱、无障碍旅行车和移动行宫的角色。

整车气质由传统美式豪华 Conversion Van、GMC 重型商用厢式车、北境猎场行营与现代大功率生活电系统共同构成。它不走现代奔驰商务舱式的冷白灯、亮黑饰板、满墙菱格与大面积镀铬，也不做裸露钢架、铆钉板和战术黑仓库风格。

摄政王的核心关键词是：

> **老钱、重载、豪华、四驱、长续航、低调、北境、猎场、移动行宫、御驾亲征。**

## 2. 上位工程原则

摄政王允许拥有复杂功能，但不允许所有功能依赖同一个脆弱的中央系统。最终工程哲学是：

> **关键系统尽量简单；系统之间彼此隔离；重要功能能够降级运行；需要时保留机械或手动备份。**

空调、暖风、生活电、水泵、通风、无障碍升降平台、照明和家具锁止均有独立控制逻辑。单个智能控制器失效，不应导致整车无法饮水、照明、换气或基本生活。

所有超过约 5kg 的设备都必须进入结构化固定系统。家具外观属于内饰问题，家具在急刹、侧翻或碰撞中是否脱离，则属于车辆安全结构问题。

## 3. 基础车辆与最终身份

摄政王的原始车辆为：

> **2020 GMC Savana 3500 Extended Cargo**

它是完整货运车身，不是 Cutaway，也不是底盘驾驶室。驾驶舱与生活舱完整贯通，保留前门、乘客侧中门和后双开门体系。

车辆在事故后以残骸为基础彻底重构。此前如何一步步改装、发生过哪些中间故障、换过哪些过渡零件，本文件暂不展开，只记录重构完成后的最终状态。

| 项目 | 最终设定 |
|---|---|
| 底车 | 2020 GMC Savana 3500 Extended Cargo |
| 车身 | 长轴完整厢式货运车身，非 Cutaway |
| 轴距 | 约 155 英寸 |
| 驾驶舱 | 与生活舱完全贯通 |
| 主要乘员 | 两名成人、一只小型英卡 |
| 额外能力 | 永久保留 Sam 无障碍登车、乘坐、转移与驾驶兼容 |

## 4. 尺寸、限高与重量口径

安装 Tufport 24 英寸固定高顶后，车身主体高度约在 2.75—2.85m 范围；加入太阳能、天窗边框和最高附件后，目标最高点控制在 3.00m 左右。

驾驶管理不按理论最低值，而固定按：

> **10ft / 3.05m 限高车辆**

处理。驾驶位长期保留明确的车高、车长、总质量与拖车状态提示。

最终重量口径为：

| 状态 | 规划质量 |
|---|---:|
| 精简日常旅行状态 | 约 4.55—4.70 吨 |
| 满油、满清水、两人、英卡与正常补给 | 约 4.70—4.85 吨 |
| 全远征模块、猎具、摄影与冬季装备 | 约 4.85—4.95 吨 |
| 整车工程设计总质量上限 | **约 5.00 吨** |

五吨是整车自身的工程设计上限，不等于每次旅行都顶着五吨行驶，也不能简单视为原厂铭牌 GVWR 或法定轴荷已自动改变。完工后必须进行空载、常态旅行、满水满油和拖车状态的四轮称重。

# 二、动力、四驱、后桥与拖车系统

## 1. 6.6L 机械增压动力

最终动力采用：

> **GM 6.6L L8T 汽油 V8 + Whipple 或同级双螺杆机械增压系统**

目标不是追求极限峰值，而是让五吨级高顶车在高原、逆风、山路和拖车时拥有充足的低转速储备。

| 模式 | 输出 | 使用定位 |
|---|---:|---|
| 巡边 Marchwarden | 约 550 马力、800 牛·米 | 日常重载、拖车、低附着与长距离使用 |
| 远岬 Foreland | 约 600 马力、880 牛·米 | 山路、高海拔、超车与主要远征模式 |
| 御驾 Regency | 约 650 马力、950 牛·米 | 短时全性能释放，不作为拖车常驻模式 |

正常使用 91 或 93 号汽油；89 号只允许在无法获得高标号燃油时，以最保守的巡边模式应急使用。

发动机配套加强中冷、机油冷却和冷却液系统。前两挡保留明显限扭，避免四驱、重载或拖车状态下用峰值扭矩冲击变速箱、分动箱和后桥。

## 2. 强化 6L90 六速自动变速箱

变速箱锁定为：

> **强化 GM 6L90 六速自动变速箱**

保留传统怀挡，符合 Savana 驾驶舱气质，也能给前排中部和无障碍转移留出空间。

强化方向包括离合器组、阀体、液力变矩器、油底壳、外置油冷和温度监测。标定以重载耐久为先：

- 校正 LT265/75R16 胎径与 4.10 齿比；
- 减少五、六挡之间频繁游猎；
- 禁止低转高负荷拖挡；
- 长下坡更积极降挡并保持发动机制动；
- 拖车状态优先控制变速箱油温；
- 保留热保护、扭矩管理与故障降级能力。

御驾模式不意味着关闭保护系统。

## 3. Quigley 标准高度四驱

四驱采用经典方案：

> **Quigley GM 标准高度 IFS 四驱转换体系**

主要结构包括 GM 扭杆式独立前悬、GM K 系列前差速器与半轴、前桥断开机构、GM NP261 / 261 系列两速分动箱、机械式分动箱挡杆，以及 2H、4H、4L 三种基本状态。

它不采用夸张升高，也不追求岩石攀爬。任务是可靠处理积雪、泥地、碎石路、猎场林道、沙地、小溪和北境非铺装路面。

## 4. AAM 9.5 英寸强化后桥

后桥保留原车 AAM 9.5 英寸桥体，在此基础上全面强化：

- 桥壳和桥管探伤、校正与必要加强；
- 强化半轴、轴承、油封与轴端；
- 高强度主减速齿轮；
- 强化钢板弹簧座与后桥安装点；
- 加大容量差速器盖；
- 后桥油温传感器；
- 磁性放油螺栓与高品质合成齿轮油。

后差速器升级为：

> **Eaton Detroit Truetrac 齿轮式强化 LSD**

按 AAM 9.5 英寸、33 花键体系理解。相比摩擦片限滑和过于粗暴的自动机械锁止，Truetrac 更适合高速巡航、冰雪、拖车和轻中度越野。

前后桥统一齿比：

> **4.10:1**

4.10 与 31.6 英寸左右轮胎、6L90 六挡和机械增压 6.6L 组合后，兼顾高速巡航、五吨级起步、山路和拖车。3.73 偏高，4.56 又会牺牲过多高速转速和安静性。

## 5. 悬挂、轮胎与制动

后悬挂采用强化多片钢板弹簧、重载减振器、加强防倾杆和后轴辅助空气包。空气包用于调平、控制左右姿态和拖车时抑制车尾下沉，但不把空气包本身视为提高法定后轴额定值的证明。

前悬配合 Quigley 体系升级扭杆、减振器、衬套和定位参数。制动系统加强散热、摩擦材料和温度管理。

轮胎锁定为：

> **LT265/75R16 Load Range E**

外径约 31.6 英寸。优先选择 Michelin Defender LTX M/S 或 Michelin Agilis CrossClimate。轮毂采用简洁的 16 英寸全锻造轮毂。

车外必须携带一只完整带轮毂备胎，不接受只带外胎。

## 6. 燃油与续航

最终按：

> **57gal 级主油箱 + 3gal RotopaX，约 60gal 总储油**

理解。3gal RotopaX 隐藏在加深的备胎软罩内部，通过独立金属固定座、锁扣和隔板固定，不靠软罩承重。

| 工况 | 规划油耗 |
|---|---:|
| 南部平直高速 | 约 11—12mpg |
| 普通综合公路 | 约 10—11mpg |
| 落基山、高原和山路 | 约 8.5—10mpg |
| 满载、逆风、拖车和复杂山区 | 约 8—9.5mpg |

60gal 理论续航约 770—1160km；路线规划按 80%—85% 可用燃油计算，安全续航约 650—950km。

## 7. Bronco 与小拖车能力

摄政王拥有正式拖车系统，主要承担两类任务：

1. 四轮落地平拖四门 Ford Bronco 副车；
2. 偶尔拖带普通小型有制动拖车。

Bronco 负责摄政王不应进入的最后几公里。平拖系统包括专用车架底座、重型拖杆、辅助制动、Breakaway 紧急制动开关、尾灯线束、维持充电线和独立安全绳。

普通拖车的常用范围约 1.5—2.5 吨，机械设计上限按 3 吨控制。接近 3 吨时，拖车必须具备独立制动，并使用重量分配和防摇摆系统；同时摄政王自身不得处于五吨极限满载状态。

后拖接收器采用：

> **定制车架级 2.5 英寸 Class V 接收器**

结构参考 Draw-Tite Titan 45507 一类重载接收器，但不裸露安装。主横梁隐藏在低矮 Conversion Van 后下包围内部，中央方口通过齐平隐藏盖板露出。

不拖车时，车尾只看到一块与下包围平齐的盖板；拆开后才能接触接收器、插销、安全链点和 7 针接口。接收器绝不为了绕过低下包围而整体吊低。

# 三、高顶、车身、隔音与老钱风内饰

## 1. Tufport 24 英寸固定高顶

高顶锁定为：

> **Tufport 长轴专用 24 英寸固定玻璃钢高顶**

名义加高约 61cm，采用一体式玻璃钢结构。前部缓坡，中后部空间完整，顶部保留合理排水弧度，不做升降顶，不开夸张高顶侧窗。

Tufport 的价值不只是增加高度，还包括更好的内部空间、加强结构基础和 NVH 起点。切顶后必须重建车顶横向刚度，使用完整法兰、加强框架和结构胶体系。

## 2. 车内宽度与地板

完成隔音、隔热和侧墙内衬后，纯平净宽按约 175cm 规划。中腰部如果顺着原车弧度分段制作软包和柜体，局部宽度可达到约 182cm。

地板采用波纹找平、局部隔振、闭孔保温层、承重板和高耐磨面层。地面使用高级耐脏橡胶、Marine Vinyl 或高耐磨复合材料，上铺可拆洗短毛地垫。

## 3. 轻量化高 NVH 分层

高顶和车身统一采用轻量化分层隔音、隔热与内衬方案。

大面积空鼓区域只局部铺设丁基止振板，覆盖约 25%—40%，重点处理板面中心和容易鼓动的位置，不满铺。

筋条之间填充 25—40mm 疏水聚酯棉或 3M Thinsulate。金属筋和玻璃钢加强件表面增加薄闭孔泡棉，削弱热桥并避免内饰板直接接触结构件产生异响。

面层采用 3—5mm 轻质复合板或薄胶合板跨加强筋铺设。面板与骨架之间加入毛毡、薄橡胶或闭孔泡棉垫。

车顶明确不满铺重型隔音毡或 MLV，避免增重和抬高重心。

## 4. 传统美式老钱风软包

内饰走传统美式豪华 Conversion Van 方向，而不是现代奔驰商务舱。

主色为深烟草棕、深马鞍棕或暖灰褐。大面积使用光面高品质汽车级软皮，中央人体接触区低调打孔，同色缝线配少量细滚边。

扶手、侧墙和柜体边缘做厚实但克制的圆角软包，可搭配少量深色胡桃木或暗色木纹饰板。

明确不使用密集菱格、大片钢琴黑、冷白灯带和大面积亮铬。

## 5. 外观与车顶设备

车身以渐变墨绿金属漆为主体，搭配传统 Conversion Van 横向长条纹、克制的低矮侧裙和后下包围。

左侧可用假窗或深色反光装饰条平衡货运车侧面的视觉长度；右侧设置实用踏板，并与乘客侧无障碍设备整体协调。

车顶只保留必要的低矮设备：约 900W 固定太阳能、约 600×800mm 深蓝黑固定天窗或星空天窗、低矮布线盒和维护通道。没有全尺寸越野顶架，也没有显眼的大号车顶 Maxxair。

尾部一侧为完整备胎与隐藏油桶，另一侧为空调外机与可拆窄梯，中央下方为隐藏式 2.5 英寸拖车接收器。

# 四、48V 电力、太阳能、空调与通风

## 1. 20.48kWh 生活电系统

核心电池为：

> **EG4 51.2V 100Ah / 5.12kWh × 4**

总标称容量：

> **20.48kWh，口语按 20 度电理解。**

四块电池两两上下叠放、两组并排，横向总占用约 1m，总重约 181kg，安装在右侧沙发床下方、靠近后轴前方的低位结构舱内，并与车辆左侧满水后约 136kg 的主清水箱形成左右质量平衡。

同一区域分舱布置继电器、主熔断、母排、逆变/充电器、行车充电设备和维护断路器。电池托架为独立刚性金属结构，锚定车体横梁和加强板；高压电气区与狗窝、水路、保险柜、沙发软包之间均设置阻燃实体隔板、维护间隙、独立维护盖板和通风路径。

| 系统 | 设定 |
|---|---|
| 电池 | EG4 51.2V 100Ah ×4 |
| 总容量 | 20.48kWh |
| 逆变/充电器 | 6000W 级纯正弦波 |
| 岸电 | 50A 优先，兼容 30A 与 20A 转接 |
| 行车充电 | 约 3kW 级 48V 行车充电 |
| 固定太阳能 | 约 900W |
| 停车外挂太阳能 | 600W 折叠板 |
| 独立发电机 | 不安装 |

固定太阳能与外挂太阳能尽量使用独立 MPPT 输入，避免不同角度、阴影和电压特性互相拖累。

## 2. 600W 停车外挂太阳能

600W 折叠太阳能板只在停车时展开，可放置在地面调整朝向，也可临时安装在车侧专用挂点。行车时完整收纳。

可拆尾梯用于安装和调整，但太阳能板不能依赖梯子本体承受风载。

## 3. 低噪音 48V 分体空调

生活区空调采用：

> **48V 直流低噪音分体式空调 + 隐藏式风管内机**

制冷量约 9000—11000BTU。外机安装在尾部独立车架级或加强结构摆臂上，与车身之间使用减振软连接。

内机隐藏在高位柜体或顶柜中，通过 2—3 个短风管高位送风口覆盖二排、餐厨和生活区。柜体下方或沙发低位设置中低位回风。

驾驶舱行驶时使用原车空调；尾部约 50cm 工具区不作为主要制冷区。

空调稳定功率约 400—900W。普通夏季每天约 6—12kWh；凤凰城极端暴晒或宠物安全舱长时间运行时，可能超过 15kWh。

## 4. 尾门上方换气系统

主换气风扇采用：

> **黑色 MaxxFan Dome 或同类低矮侧壁排风扇**

安装在尾门上方的固定玻璃钢高顶后围，而不是反复开合的尾门门板上。从车头和大多数侧面基本看不到。

主空气路径为：

> 前部低位可控补气 → 二排 → 餐厨 → 生活区 → 尾部高位排出

尾部工具区有时会被隔帘封闭，因此隔帘上方设置永久隐藏旁通风道。隔帘关闭后，主生活区仍能通过帘盒或顶柜内的风道直接进入尾部排风系统；工具区另保留较小支路或格栅。

空调运行时，主排风原则上关闭，只在做饭、排湿、快速换气和清除异味时短时使用。

## 5. 柴油暖风

冬季暖风采用：

# 五、车内 3.75m 空间的精确分区

本章只记录设备的空间位置、占用尺度、展开方向与相互避让关系；型号、性能、用途和安全结构统一在后续专业章节展开。

## 1. 统一方向与尺度口径

以下左右方向均以车辆正常前进方向、驾驶员坐姿为基准。从前排座椅后缘到后门内侧，连续可用生活空间约 3.75m，分成三个互不重叠的连续区域：

| 纵向范围 | 区域 | 长度 |
|---|---|---:|
| 0—1.40m | 餐厨与二排区 | 约 1.40m |
| 1.40—3.25m | 主生活、影音与床铺区 | 约 1.85m |
| 3.25—3.75m | 尾部工具与卫生区 | 约 0.50m |

## 2. 第一分区：0—1.40m 餐厨与二排区

车辆右侧、乘客侧中门内侧设置唯一的二排座椅，安装在重载滑轨与旋转机构上；座椅后方与右侧沙发床前端之间，是可抽出英卡狗窝向前露出的部分。

车辆左侧前部依次为水槽、备餐板和上方橱柜，水槽下方为常用 5gal 饮用水瓶；左侧后部、接近本分区末端的位置竖向安装全尺寸冰箱。乘员坐在右侧二排座椅上时，左手一侧是冰箱，左前方是水槽与备餐板。

乘客侧中门下方为隐藏式轮椅升降平台。平台升起后，轮椅落位于二排与主生活区交界处；中门、中央通道、二排座椅、厨房柜体和冰箱共同让出座椅后移、旋转及轮椅转向包络。

## 3. 第二分区：1.40—3.25m 主生活区

第二段长度约 1.85m。左右关系正式锁定为：

> **车辆左侧是电视/影音墙及其下方的 136L 异形清水箱；车辆右侧是沙发床及其下方的四块 EG4 电池和大功率电气系统。**

此前任何相反的左右描述全部作废。

右侧沙发床沿纵向约 1.85m，睡眠时向左横向拉出至约 1.75m 宽；床下为电池和大功率电气分舱，沙发床前端下方嵌入可抽出狗窝的主体部分。

左侧电视与电子墙集中布置电视、前置音响、主机、弱电箱、生活 HUD 和电子舱；136L 主清水箱位于其下方，横向占宽约 45cm，并顺应轮拱和侧壁结构成型。影音功放设备位于电视墙低位或床下、邻近水箱但完全隔离的通风舱内。

车辆右侧上部和后段布置环绕扬声器与自行车壁挂系统，并与沙发坐姿和主床展开包络错开。靠近尾舱之前的高位设置吊顶衣橱；主生活区顶部设置两只天空声道和一套可收纳的单人吊床式上铺，收起后避开天窗、空调风口和主要站立空间。

## 4. 第三分区：3.25—3.75m 尾部工具与卫生区

尾部纵向约 50cm，左右各设一组高效柜体，中间保留约 80cm 通道并直达后双开门，允许人员站在车内操作，也便于从车尾取出工具、淋浴设备、冷藏箱和营地装备。

一侧为卫生、衣物和鞋靴柜，另一侧为工具、猎具、烧烤、渔具、冷藏与恢复装备柜。热水器的收纳位、挂点和使用接口，以及柴油暖风主机、风道接口和维护位置均集中在此；柴油箱位于底盘下方。

尾部工具区可由隔帘与生活区分开，隔帘上方接入第四章所述永久隐藏旁通风道。

# 六、水系统、厨房与卫生系统

## 1. 三层饮水体系

摄政王的水系统分为三个层级。

### 136L 主清水箱

主清水箱最终锁定为：

> **136L 定制异形清水箱**

它安装在车辆左侧电视下方，不采用规整长方体，而是针对轮拱拱起、侧壁弧度和底部不规则结构定制成型。横向宽度约 45cm，纵向和高度随轮拱与设备空间变化，通过不规则外形利用电视墙下方的低位死角。

水箱用于洗漱、洗澡、基础烹饪和英卡日常饮水，满载约增加 136kg 质量。箱体由多方向金属限位框架固定到车体结构，并设置检修口、排空口、通气、液位传感器和防晃动设计。

清水箱与影音电子舱之间必须具有防水隔离、泄漏导流和明确维护边界；任何功放、弱电箱或插座都不能直接位于无防护的水路接头下方。

### 水槽下 5gal 方形饮用水瓶

该饮用水瓶使用独立小泵，主要用于直接饮用、咖啡和精细烹饪。

### 额外 5gal 备用饮用水瓶

备用瓶不与主水箱合并，可作为跨境或偏远地区的饮水余量，也可在主饮用水瓶清洗、故障、污染或水质存疑时替换使用。

因此整车总可携带水量为：

- 主清水约 136L；
- 常用饮用水约 18.9L；
- 备用饮用水约 18.9L。

三者合计约 174L，但正常旅行不一定每次全部装满。

## 2. 灰水系统

灰水箱继续按约 80L 规划，接收水槽、洗漱和外置淋浴的合规排水。灰水箱设置液位传感器、排空阀、通气和防冻考虑。

## 3. 餐厨设备与收纳

水槽上方橱柜收纳餐具、杯具、调味品和日常小电器。厨房设备包括电磁炉、微波炉、咖啡机、空气炸锅、基础炊具和抽拉备餐板。

大功率设备可同时存在，但在 30A 岸电或电池低电量状态下需要功率管理。所有电器都有固定仓和机械锁扣，不能只靠防滑垫留在台面上。

## 4. Sharp 全尺寸冰箱

冰箱锁定为：

> **Sharp SJB1255GS 120V 家用冰箱**

冰箱高约 184.9cm、宽约 59.9cm、深约 68.6cm，总容量约 326L。它使用独立底部承重托盘、左右限位架、顶部防倾倒横梁、行车门锁和背部散热风道。

## 5. 热水与便携马桶

Camplux 5L 级便携燃气即热热水器的收纳、挂点和使用接口位于尾部工具区。使用时必须打开后门，将热水器转移到车外通风位置，不得在密闭车内点火。

典型用水量：

| 用途 | 水量 |
|---|---:|
| 冲脚、泥、狗爪 | 3—5L |
| 极简一人洗澡 | 5—8L |
| 普通简单洗澡 | 10—15L |
| 两人简单洗澡 | 20—30L |

便携马桶采用 Thetford Porta Potti 565E / 565P。它服务夜间、长途和应急，不设置固定黑水箱。

## 2. Sam 无障碍配置

乘客侧中门下方安装 BraunAbility UVL 级 Under Vehicle Lift 隐藏式轮椅升降平台。平台收起后隐藏在车底密封盒体中，具备电动液压运行和手动应急泵。

中门内侧保留平整轮椅落位和转移区。地板设置 Q’Straint 级 L-Track 与四点式轮椅固定系统。二排座椅、厨房柜体和冰箱必须共同服从中门轮椅转向包络，不能把通道挤成只适合正常步行的狭缝。

Sam 的主要使用逻辑是：轮椅从中门进入，在二排与生活区交界处停靠或完成转移，再转移到驾驶座或二排座椅。

驾驶席 De Leon 可后移并向内旋转约 150°，辅助轮椅转移。驾驶舱保留可拆或可旁路的手控加速、制动、方向盘旋钮和易触及开关，普通驾驶者仍能使用原厂踏板。

## 3. 狗窝与无障碍让位逻辑

英卡狗窝采用可抽出的低位模块，而不是完全独立摆在二排座椅后方。狗窝主体一部分嵌入右侧沙发床前端下方，另一部分向前露出到二排 Qualitex De Leon LX 座椅后方，使英卡日常能够贴近二排乘员并看到主生活区。

尺寸约 80 × 55 × 40cm，采用开放式软窝、可拆洗底盘和良好通风。底托使用带机械锁止的抽拉或快拆结构，行车时可靠固定；狗窝与电池、水路、保险柜和工具舱之间均有实体隔板。

Sam 使用轮椅时，二排座椅有时需要沿滑轨彻底后退，以释放中门升降平台后的轮椅转向与落位空间。此时先将狗窝整体抽出或移走，让狗窝不再占据座椅后移包络和无障碍通道。完成转移或恢复日常布局后，再将狗窝推回并重新机械锁止。

另配 60 × 45cm 或 70 × 50cm 低矮狗厕所托盘，只处理夜间和应急排尿。

## 4. 主床、吊床与睡眠容量

右侧沙发床日常作为双人沙发和主要观影座位，睡眠时从右侧向左横向拉出，形成约 1.85m × 1.75m 的主床，基本覆盖车内可用横向宽度。它不是窄式单人床，也不再按旧版 1.5m 宽 Queen 口径理解。

主生活区顶部设置一套高强度、可收纳的单人吊床式上铺。吊床平时卷收或折叠在顶侧，不遮挡高顶空间、天窗和空调风口；使用时连接到车体加强结构，而不是薄内饰板或玻璃钢表皮。吊床只按一名成人设计，主要作为林峻的应急睡位，不按双人吊床计算。

1.85m × 1.75m 主床的常态舒适使用仍以两人为主，但在小说中的极限过夜状态下，可以让林漪、Nico 与沈嘉宁三名女性并排挤睡；两名女性与林峻三人也可以勉强同睡主床。四人同时过夜时，优先采用三名女性睡主床、林峻睡单人吊床的安排，舒适性明显好于让三人中包含林峻挤在主床上。

因此，摄政王的常态舒适睡眠人数仍为两人，临时可睡三人，极限应急状态下可睡四人，但第四个睡位来自单人吊床，而不是让吊床承载两人。

## 5. 吊顶衣橱与高位载荷

靠近尾部工具区之前，顶部设置一只吊顶衣橱，主要收纳外套、轻薄保暖层、帽子、手套、雨具和睡眠用品。

衣橱使用正向机械锁扣和轻量化柜体，避免把大量重物堆到高位。

## 6. 家具与柜门锁止

所有柜门、抽屉和设备使用正向机械锁止：

- 柜门使用双动作机械锁扣；
- 抽屉使用重载锁止滑轨；
- 冰箱设置上下双重行车锁；
- 电视设置独立运输锁；
- PC、AVR 与音响安装在金属框架中；
- 水箱设置多方向限位；
- 电池托架连接车体横梁；
- 主保险柜锚定车体加强板；
- 磁吸、软关闭铰链和家具摩擦力不属于行车锁止。

收纳时拆前轮、转把、拆靠墙一侧脚踏、前叉固定并上翘斜挂。自行车挂在右侧生活区后段，不能侵入沙发正常坐姿与主床展开包络。

# 十、宠物、基地与旅行路线

## 1. 英卡安全舱

英卡为小号、绝育、展示系母 English Cocker Spaniel。正常饮水约 0.5—0.9L/天，热天、运动和沙漠环境可达 1.0—1.5L/天，远征按 2L/天预算。

停车独处依靠 48V 空调、20.48kWh 电池、摄像头、温度传感器和远程报警，但不以车内空调替代长时间托管。

## 2. 主堡与北境行营

Phoenix / Mesa / East Mesa 主堡定位为冬春王府、多恩军械库和日常整备基地。理想基础设施为 50A RV 岸电柱、水口和 Sewer Cleanout。

Bozeman / Belgrade 北境行营定位为临冬城和北线装备仓，存放山地车、滑雪装备、厚雪靴、雪链、猎装和极寒备用件。

## 3. 主要路线

西南红岩线：Phoenix / Mesa → Sedona → Grand Canyon South Rim → Page → Zion → Bryce → Utah 红岩高原。

南洛基—黄石—大提顿线：Phoenix → Colorado / Utah → Yellowstone → Grand Teton → Idaho / Montana。

北境大线：Bozeman → Calgary → Banff → Jasper → Icefields Parkway → Whitehorse → Dawson → Dempster Highway → Tuktoyaktuk。

猎场线：Phoenix / East Mesa → New Mexico → West Texas → Oklahoma → Arkansas。

Baja 与南美路线进入墨西哥前，按第九章规定执行枪弹清零。后期跨洲路线为 Phoenix → 墨西哥 → 中美洲 → 海运绕过 Darién Gap → 秘鲁 → 智利 → 巴塔哥尼亚 → Ushuaia → Buenos Aires 方向。

# 十二、最终定义

> **摄政王·远岬是一台由 2020 GMC Savana 3500 长轴完整货运车事故残骸彻底重构而成，采用 6.6L 机械增压 V8、强化 6L90 六速自动、Quigley 标准高度四驱、AAM 9.5 英寸强化后桥、4.10 齿比、Tufport 24 英寸固定高顶、20.48kWh EG4 电池、1500W 级总太阳能、Sharp 全尺寸冰箱、136L 左侧异形清水箱、三张 Qualitex De Leon 座椅、完整无障碍系统和精确三分区内装打造的五吨级传统美式豪华远征行宫。**

它的 3.75m 主舱不是模糊堆叠的房车空间，而是一套明确的纵向结构：前 1.40m 是二排与餐厨，中间 1.85m 是横向展开的主床、影音与设备核心，最后 0.50m 是左右柜体与 80cm 中央通道组成的工具和卫生区。

它能在城市中补给，能沿州际公路长时间巡航，能平拖四门 Bronco 抵达猎场，能进入西南红岩与落基山，也能从 Bozeman 一路北上 Whitehorse、Dempster 和北冰洋。

它不是为了证明一辆 Van 能堆进多少设备，而是让两个人、一只英卡，以及完整的生活、影像、猎场、影音和无障碍系统，在沙漠、雪线、国家公园与漫长公路之间长期而体面地生活。
