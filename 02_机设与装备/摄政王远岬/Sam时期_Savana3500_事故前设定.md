# Sam时期｜GMC Savana 3500事故前设定

> 状态：当前正式事故前基线。用于第一、二卷中Sam仍拥有并使用Savana时的车辆描写。事故后重构完成的最终形态，以《摄政王远岬_专项总设定.md》为准。

## 一、车辆定位

- 基础车：2020 GMC Savana 3500 Extended Cargo，完整长轴货运车身。
- 车主与主要使用者：Sam Mercer。
- 使用阶段：2020年购入并逐步完成早期房车化，持续使用至2023年春事故。
- 定位：适合两至四人短途旅行、北亚利桑那高原、公路露营和一般林道后勤的实用型高顶房车，不是豪华Conversion Van，也不是后来的重型远征行宫。
- 工程原则：结构清楚、设备有明确用途、关键功能能够单独工作、故障后容易隔离和现场处理。

## 二、车身、动力与通过性

- 保留原厂后驱动力与传动体系，不进行Quigley四驱转换。
- 不做夸张升高；使用重载公路／全地形轮胎、完整带轮毂备胎和基础悬挂整备。
- 随车携带气泵、补胎工具、轮挡、铁锹、牵引带、树带、手动牵引设备及基础恢复工具，但不把恢复装备当成进入危险道路的许可。
- 安装固定式玻璃钢高顶，满足车内站立、换衣、做饭和整理装备；具体品牌与名义加高尺寸暂不锁死。
- 车顶只布置必要的太阳能与通风设备，不安装大型行李架、车顶空调或装饰性附件。

## 三、乘员与车内布局

- 前排保留驾驶席与副驾驶席。
- 驾驶舱后方设置一张经过可靠结构固定的前向双人座椅，全车共四个合法乘员位。
- 乘客侧中门及其内部通道保持完整，不用家具或设备长期侵占上下车路径。
- 中前部左侧设置短厨房柜，包括小水槽、简单操作台、45—60L 12V压缩机冰箱及少量餐厨收纳。
- 炉具以便携式为主，收纳后带到车外使用；不安装固定大容量丙烷系统。
- 后部左右设置纵向箱式长凳，白天保留中央通道，夜间以可拆连接板和床垫拼成双人床。
- 长凳下方分别容纳生活电、水箱、暖风、工具、毯子、急救与露营装备；所有设备均设置独立检修口和机械锁止。
- 不安装电动沙发床、全尺寸家用冰箱、固定大电视、私人影院或复杂智能中控。

## 四、生活电系统

Sam时期采用简单的12V生活电架构，与事故后摄政王的48V系统明确区分。

| 系统 | 事故前设定 |
|---|---|
| 电池 | 12.8V 200Ah LiFePO4 |
| 标称电量 | 约2.56kWh |
| 逆变器 | 1000—1500W级纯正弦波 |
| 行车充电 | 12V DC-DC行车充电 |
| 固定太阳能 | 约300—400W |
| 岸电 | 简单岸电充电器与外接接口 |
| 独立发电机 | 不安装 |

- 逆变器主要服务笔记本电脑、相机电池、充电器和短时小功率120V设备，不承担长期电烹饪或空调负载。
- 冰箱、照明、水泵、通风和柴油暖风尽量直接使用12V电源，减少不必要的逆变损耗。
- 电气分路清楚，保险丝、断路器和线缆均有标签；不依赖单一触控屏管理全车功能。

## 五、水、暖风与通风

- 车内清水容量约50—60L，主要服务饮水、简单烹饪和洗漱。
- 设置约20—30L灰水容量或等效可拆排水方案；不安装固定黑水箱。
- 使用简单12V水泵，并保留能够绕过电泵的基础取水方式。
- 安装2kW级柴油暖风，用于高原寒夜与冬季短途使用。
- 高顶设置简单可靠的排风／换气设备；没有生活区空调。
- 使用便携马桶或应急卫生方案，不建立完整固定卫生间。

## 六、车辆气质与后续关系

Sam时期的Savana内部以耐用板材、开放检修口、机械锁扣和清晰标签为主。它能够让人站立、做饭、睡觉、取暖并处理一般道路突发情况，但不会为了舒适而加入大量高功率电器和复杂家具。

事故后，林峻以残骸为基础重建摄政王。最终车辆的48V生活电、20.48kWh电池、四驱、全尺寸冰箱、影音系统、无障碍结构和五吨级远征能力均属于事故后的重新设计；Sam时期只提供原始车体、早期房车化经验及简单可靠的工程起点。
